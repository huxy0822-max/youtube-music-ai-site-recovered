YouTube 视频发布助手（多模型 API）

输入内容类型 + 上传 YouTube 受众截图，自动生成标题、简介、两套标签、缩略图英文指令（图中文字要求繁体中文）。

1) 模型与 API 配置

文本 API

OpenAI 兼容默认填 DeepSeek，可手动改成任意兼容地址。

生图 API

会与文本API一起保存到当前模板名。

2) 内容输入

内容模板仅保存内容输入/生成参数/提示词/标题库，不包含API配置。
默认繁体中文，可切换为 YouTube 常用语言。选哪个语言，文案与封面文字就按哪个语言输出。
把截图拖到这里，或直接 Ctrl/Cmd + V 粘贴截图
“选择文件”是备选，日常直接拖拽/粘贴就行。

3) 生成参数（可自定义）

0/5000 标题库会和提示词一样，直接在前端可见可编辑。
当前限制：标题80-95字，简介约300字，标签10-20，默认3标题/1简介/3缩略图。可在上方自定义。
就绪

4) 结果（每块都可一键复制）

默认会生成 3 标题 + 1 简介 + 2 套标签 + 3 缩略图指令。你改了数量后会按你配置生成。
